# 5 Fitur Komunikasi Pasien & Masyarakat

Dokumen ini berisi usulan fitur agar layanan dapat berkomunikasi dua arah dengan pasien maupun masyarakat.

## 1) Forum Ide & Saran Publik
- Masyarakat dapat mengirim ide peningkatan layanan.
- Admin bisa memberi status: `baru`, `ditinjau`, `diimplementasikan`.
- Ide populer ditampilkan berdasarkan dukungan (upvote).

## 2) Konsultasi Chat Terjadwal
- Pasien melakukan booking slot chat dengan petugas.
- Integrasi notifikasi pengingat sebelum sesi dimulai.
- Riwayat percakapan tersimpan untuk tindak lanjut.

## 3) Q&A Komunitas dengan Moderasi Medis
- Pertanyaan umum pasien dapat dijawab tim edukasi.
- Jawaban tervalidasi diberi badge "ditinjau tenaga kesehatan".
- Pertanyaan berulang otomatis direkomendasikan ke FAQ.

## 4) Pelaporan Cepat Keluhan Layanan
- Form singkat untuk keluhan fasilitas/administrasi.
- Setiap laporan memiliki nomor tiket dan SLA respons.
- Dashboard admin menampilkan prioritas berdasarkan urgensi.

## 5) Survei Kepuasan & Polling Program
- Survei berkala untuk kualitas layanan, kenyamanan, dan akses informasi.
- Hasil ringkas ditampilkan transparan untuk publik.
- Polling tema edukasi membantu menentukan konten berikutnya.
